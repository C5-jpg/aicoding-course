# AI 编程课程资源合集

本仓库收集了斯坦福大学 AI 相关编程课程的学习资源，专注于 AI 辅助编程和现代软件开发。

## 课程列表

### 1. CS 224G: Building & Scaling LLM Applications
- **定位**: AI 时代的软件工程 / LLM 工程化
- **官网**: https://cs224g.stanford.edu/
- **描述**: 探索 LLM 应用边界的项目课程，学习 LLM 的能力和局限性

### 2. CS 146S: The Modern Software Developer
- **定位**: AI 改造的软件开发全流程
- **官网**: https://themodernsoftware.dev/
- **描述**: 展示现代 AI 工具如何提高开发者生产力

### 3. CS 185: Coding with LLM Assistants
- **定位**: LLM 编程助手 + 人机协作
- **描述**: 研究视角探索 LLM 辅助编程

### 4. CS 106AI: Programming with AI
- **定位**: AI 辅助的编程入门
- **官网**: https://cs106a.stanford.edu/
- **描述**: Python 编程基础与 AI 应用

### 5. Signals and Systems (教材)
- **作者**: Oppenheim & Willsky
- **描述**: 信号与系统经典教材（下载受限，建议从图书馆获取）
