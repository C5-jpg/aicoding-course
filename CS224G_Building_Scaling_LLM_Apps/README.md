# CS 224G: Building & Scaling LLM Applications

## 课程信息

- **课程名称**: Apps with LLMs Inside
- **机构**: Stanford University
- **官网**: https://cs224g.stanford.edu/
- **课程定位**: AI 时代的软件工程 / LLM 工程化

## 课程大纲

本课程探讨围绕 LLM 构建的应用空间，通过实践项目学习 LLM 的能力和局限性。

### 主要主题
- LLM 应用架构
- 提示工程 (Prompt Engineering)
- 上下文工程 (Context Engineering)
- LLM 产品开发
- 数据策略
- 敏捷开发与 LLM
