# CS 146S: The Modern Software Developer

## 课程信息

- **课程名称**: The Modern Software Developer
- **机构**: Stanford University
- **官网**: https://themodernsoftware.dev/
- **课程定位**: AI 改造的软件开发全流程

## 课程描述

本课程展示现代 AI 工具不仅能提高开发者生产力，还能使软件工程更加民主化。学生将掌握传统软件工程挑战背后的理论，以及当今解决这些问题的尖端 AI 工具。

## 主要主题

- AI 编程范式
- 现代开发工具
- 自动化测试
- 智能文档
- 安全漏洞检测
- CI/CD 与 AI 集成

## 课程目标

- 掌握现代开发工具
- 理解 AI 辅助编程
- 学习自动化测试和部署
- 探索新兴软件趋势

## 学习资源
